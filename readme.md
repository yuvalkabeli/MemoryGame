# MemoryGame
### Installation

1.Clone the repo
   ```sh
   git clone https://github.com/yuvalkabeli/new-dictionary-app.git
   ```
2.  Install NPM packages
   ```sh
   npm install
   ```
3.Input the start command to start local server
```sh
   npm run start
   ```
4.enter the game through: http://localhost:8000
